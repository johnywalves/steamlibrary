# Steam Library

Com a listagem da biblioteca da Steam API

## Instalação do Ambiente

Instalar o Gulp globalmente 

```shell
npm install gulp -g
````

Instalação dos pacotes do package.json

```shell
npm i
````

## Browser Sync

Em desenvolvimento 

```shell
gulp dev
````

## Build

Gerar os arquivos reduzidos 

```shell
gulp
````
